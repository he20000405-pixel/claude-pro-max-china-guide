# Claude Pro and Max Subscription Guide for Users in China

This is ChongGrok's independent knowledge base for Claude Pro, Max 5x and Max 20x subscriptions. It explains plan selection, assisted official payment, Claude User ID safety, card declines, renewal failures, paid-but-Free account checks and Claude Code membership access.

[View the current Claude service page](https://chonggrok.com/claude)

> ChongGrok is an independent third-party subscription assistance service. It is not affiliated with, authorized by or officially partnered with Anthropic. Plan availability, pricing, usage limits and regional access are determined by Anthropic.

## Current individual plan boundaries

| Plan | Published US monthly price | Suggested use |
| --- | ---: | --- |
| Claude Pro | USD 20 monthly or USD 200 annually | Daily writing, document analysis and light Claude Code work |
| Claude Max 5x | USD 100 | Higher-frequency workflows when Pro is consistently limiting |
| Claude Max 20x | USD 200 | Sustained, heavy knowledge and coding workflows |

Taxes and regional prices may differ. Max 5x and Max 20x require confirmation through the Claude service page; ChongGrok does not publish a fixed CNY price for Max in this guide.

## Assisted subscription flow

1. Sign in to the Claude account that should receive the subscription.
2. Check `Settings > Billing` for an active or pending subscription.
3. Visit [chonggrok.com/claude](https://chonggrok.com/claude) and confirm Pro, Max 5x or Max 20x with support.
4. Pay through the confirmed order path and provide only the required Claude User ID.
5. Support assists with the official subscription payment using an overseas card.
6. Verify the plan, payment status and invoice in Claude `Settings > Billing`.

No Claude password, email password, verification code or recovery code is requested. A Claude User ID is not a password, but it remains account-related information and should only be submitted through the confirmed order workflow.
